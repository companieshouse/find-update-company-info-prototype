What happens if you're added to an authorised agent account

You’ll be able to use services for the authorised agent. This includes:

  • telling Companies House you’ve verified someone’s identity
  • filing on behalf of clients

You'll be added as an account owner

This means you'll be able to view, add and remove other users from the account.

You’ll be able to update the authorised agent’s details on the account. You, or another account owner, must do this within 14 days if there are any changes to the details we hold about the agent.

You can also close the account if the agent no longer needs to be registered as an authorised agent with Companies House.
