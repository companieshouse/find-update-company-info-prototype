Someone wants to add you to an authorised agent account

If you accept, we'll add you to as a user for MORRIS ACCOUNTING LTD.

You’ll be able to use services for the authorised agent. This includes to:

  • tell Companies House you’ve verified someone’s identity
  • file on behalf of clients

You’ll be added as a standard user, which means you will not be able to add or remove other users from the authorised agent account.

What roles users can have for an authorised agent

There are 3 roles users can have for an authorised agent:

  • account owners – can update the authorised agent’s details, and add and remove all users
  • administrators – can add and remove administrators and standard users
  • standard users – cannot add and remove other users
