What happens if you're added to an authorised agent account

You’ll be able to use services for the authorised agent. This includes to:

  • tell Companies House you’ve verified someone’s identity
  • file on behalf of clients

James Morris has requested to add you as an administrator. This means you'll be able to view other users who have been added to the account.

You'll be able to add and remove other administrators and standard users, but not account owners.

What roles can a user have for an authorised agent?

When someone adds a user to an authorised agent account, they can choose which role they want the user to have. If you're added as an:

  • account owner – you can update the authorised agent’s details, and add and remove all users
  • administrator – you can add and remove administrators and standard users
  • standard users – you cannot add and remove other users
